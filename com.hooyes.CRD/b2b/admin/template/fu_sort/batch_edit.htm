栏目修改
以下每个项目有一个多选框,如选中,则修改其属性,如不选中,则不做修改
当前栏目:   $name,
修改后属于哪个分类: $sort_fup
本栏目发表内容默认属于哪个模型: $module_id
是否在首页显示本栏目文章(不包括子栏目文章): 显示 不显示
允许投稿的用户组,（不选择，则无权限.然而超级管理员与栏目管理员默认有权限）: $group_post
浏览标题的用户组,（不设置，则所有用户组有权限.然而超级管理员与栏目管理员默认有权限）: $group_viewtitle
浏览标题+内容的用户组,（不设置，则所有用户组有权限.然而超级管理员与栏目管理员默认有权限）: $group_viewcontent
下载附件的用户组,（不设置，则所有用户组有权限.然而超级管理员与栏目管理员默认有权限）: $group_download
栏目版主,多个版主,请用英文逗号,隔开:
栏目模板: 头部模板: $tpl_head
底部模板: $tpl_foot
列表模板: $tpl_list
内容模板: $tpl_bencandy
栏目风格 $style_select
是否禁止在父栏目显示本栏目的文章列表
(当父栏目下有很多子栏目的时候,就没必要全部显示出来)		 显示 不显示
是否显示评论 启用评论 禁用评论
列表页文章排序方式:
列表页每页显示多少篇文章
列表页缓存时间:
(一般留空,当调用标签太多.效率慢时,才启用)		 分钟
若有子栏目,每个子栏目显示多少篇文章: 行 (留空则以默认为标准)
若有子栏目,每篇文章显示多少个字节(一汉字相当于两个字节) 个 (留空则以默认为标准)
