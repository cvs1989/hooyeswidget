会员注册设置
是否禁止注册 禁止注册 开放注册
注册时间间隔 写0不限制，单位分钟，要大于此时间才可重新注册
注册是否使用验证码 使用 不使用
禁止注册的IP（每个IP换一行）
禁止注册的用户名（每个用户名换一行）
新注册可得{$webdb[MoneyName]} $webdb[MoneyDW]
注册是否自动通过审核:如果设置不自动通过审核的话,请配置邮件服务器,以方便用户通过邮件验证进行审核 自动通过审核 未审核状态(需要邮箱验证或管理员审核)
是否允许使用同一个邮箱注册多个帐号: 同一个邮箱可以注册多个帐号 同一个邮箱不可注册多个帐号
注册时,是否让用户选择企业会员:注册成企业用户的提示语模板是/template/default/company_reg.htm 允许 不允许
注册时允许会员选择的用户组:注意:只能是系统组.管理组与升级积分大于0的用户组,都是禁止选择的,如果选择了其它,那么普通会员是必须要选择的 $reg_group
 
注意：如果你整合了论坛，以上部分功能设置会无效,因为论坛也有一个注册页面.但你可以把论坛的注册页面关闭或删除.
